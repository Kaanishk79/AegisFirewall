# Vipera Firewall v1.0.0

**Release Date:** July 2026

## Overview

Vipera Firewall is a modern Windows firewall and intrusion prevention system built with a Rust backend and a PyQt6 desktop interface.

Version 1.0.0 is the first public release and includes packet inspection, connection monitoring, DNS monitoring, process monitoring, blocklist management, and an integrated intrusion prevention system.
